Pulse

Create a complete single-file HTML showcase called "Pulse".

A modern AI startup, agent platform, and intelligent SaaS design system.

This is NOT corporate consulting.
This is NOT cyberpunk.
This is NOT generic SaaS.

The visual language must communicate:

intelligence
speed
clarity
productivity
trust
technical elegance
human-machine collaboration

Visual inspiration:

OpenAI
Anthropic
Perplexity
Linear
Notion AI
Vercel
Raycast
Cursor
Lovable
LangChain

Aesthetic direction:

* Clean light-first interface
* Optional dark-mode sections
* Soft off-white surfaces
* Deep charcoal typography
* Subtle violet, indigo, blue, and mint accents
* Gentle gradients only when useful
* Rounded but precise components
* Lots of white space
* No neon overload
* No cartoonish elements
* No fake 3D mascots

Typography:

* Modern sans-serif
* Clean product headlines
* Highly readable body text
* Mono text for prompts, model calls, logs, traces, and API-like metadata

Visual mood:

AI product launch.
Agent workspace.
Modern developer/productivity tool.
Calm but powerful.

Components to showcase:

* Hero section for an AI agent product
* Chat assistant interface
* Agent workflow builder
* Prompt/version management cards
* Model selection dropdown
* RAG knowledge base panel
* Tool/function calling timeline
* Eval results dashboard
* Trace/log viewer
* Usage and latency metrics
* Team collaboration panel
* API key/settings card
* Pricing cards
* Customer quote section
* Product feature grid
* Security and governance panel
* Integration logos/cards
* Documentation/code snippet block
* Call-to-action section

Interaction style:

* Smooth but restrained microinteractions
* Hover states with soft elevation
* Focus states that feel accessible and professional
* Subtle loading states
* Clean modals, dropdowns, tabs, filters, badges, empty states, and toasts

Keywords:

AI startup
agent platform
LLM ops
copilot
productivity
developer tools
RAG
evals
traces
workflow automation
modern SaaS
clean interface
trustworthy AI

The final result should feel like a serious AI product used by technical teams and product teams, not a toy demo.

Someone viewing the page should immediately think:

"This is a modern AI platform people could actually use at work."
